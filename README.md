## 🚀 Laboratory Assignments & Projects
This repository acts as a live log of the practical applications, tutorials, and mini-projects completed during my class sessions. Every folder represents a step forward in mastering backend systems, enterprise integration, and desktop interfaces.

* **Desktop & GUI Engineering:** Programmatic layout controls, container mappings, and component management using Swing and JavaFX.

* **Enterprise & Network Frameworks:** Progressing toward distributed computing, local persistence configurations, and dynamic web server controls as covered in the syllabus.
